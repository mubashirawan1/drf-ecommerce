# QuickConnect - Django REST Framework Ecommerce Project

## Project Overview
QuickConnect is a powerful ecommerce backend built with **Django** and **Django REST Framework (DRF)**.  
It provides APIs for managing products, categories, shopping cart, and orders.  
This backend can be connected to any frontend (React, Vue, Angular, mobile apps) for a complete ecommerce solution.

---

## Features

- **Products & Categories**: Add, view, update, and delete products and categories  
- **Shopping Cart**: Add items, remove items, and view cart  
- **Orders**: Place orders and view user-specific orders  
- **User Authentication**: JWT-based authentication (optional)  
- **Admin Panel**: Manage products, categories, orders, and users via Django admin  

---

## Technologies Used

- Python 3.x  
- Django 4.x  
- Django REST Framework  
- SQLite (default, can be replaced with PostgreSQL or MySQL)  
- Pillow (for image handling)
